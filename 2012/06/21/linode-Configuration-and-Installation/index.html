Linode配置安装小记

发表于 2012-06-21 | 分类于 爱折腾 |

　　昨天 @lifanxi 在v2ex上出手他秒到的多余的Linode，想了想，要他帮我预留了一个。

　　下面记录一下配置Linode的一些过程，备忘。

　　对于linux来说，也有一个“远程桌面连接”，但这个连接没有ui图形化系统，是纯字符/命令行的，它的默认端口是22，而连接工具则有很多中，我在win上用的是putty，Mac的回家在找找看看什么好用。

更新系统组件

　　在安装/配置系统生产环境之前，一般都需要对现有系统进行升级，以确保系统内的包/组件达到最新版本，而debian/ubuntu和centos的升级命令是不同的，详情如下：

debian/ubuntu，输入：
apt-get update
系统进行完毕后，输入：
apt-get upgrade

在linode上安装lnmp一键包，配置生产环境！

  1. 登陆VPS
  2. 安装lnmp一键包：

wget -c http://soft.vpser.net/lnmp/lnmp0.8.tar.gz
这个命令是将“lnmp一键包”下载到你的vps中。

tar zxvf lnmp0.8.tar.gz
这个命令是解压刚才的一键包。

cd lnmp0.8/
这个命令是进入到刚才解压出来的文件夹中。

./ubuntu.sh
这个命令是运行文件夹中的“lnmp一键包”安装脚本。

  1. 安装开始前，脚本会提示你输入一个域名，这个域名是用来“登录/管理”以后的phpmyadmin、ftp的，所以，这个域名最好以二级域名的形式来搞，比如: xx.abc.com

  2. 接下来会要你输入一个MySQL的root密码

  3. 按照提示，按任意键开始安装。脚本的安装大概需要20~40分钟时间，耐心等待即可，主要看你的vps的效能和网速。

  4. 安装完毕后，你的系统就已经具备php、mysql、phpmyadmin了。

  5. 安装eAccelerator，为你的php运行提速，命令是：

./eaccelerator.sh
　　这里他会问你是安装新版本的还是旧版本的，我选了新版本:)

  1. 安装ftp，命令是：

./pureftpd.sh
　　这里推荐安装pureftp，因为其具有web管理界面，比较适合初次接触linux系统的童鞋。因为你可能已经习惯了windows的图形操作界面，而不是linux的文本化操作。

　　安装FTP的时候，会再问你mysql的root密码是什么，见第四条。同时会要你输入FTP的密码和用户名，按照提示操作。

  1. 以上内容完毕之后，基本就搞定了vps的生产环境。各管理工具如下（以之前设置的二级域名为例 xx.abc.com）
