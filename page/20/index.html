用上了小黑

发表于 2008-11-16 | 分类于 爱折腾

　　和往常一样，早上起来打开了电脑看一下未读的GoogleReader，看着看着忽然电脑哗的一下，屏幕变成了蓝白相间的竖条条，键盘失去作用，死机啦，只能重启，结果重启之后就进不了系统……
　　“阿姨啊，请问我能打800免费电话么？”，我没有固话，只好跑到了建行营业厅，最后好心的阿姨帮我在建行营业点拨通了Hp的800，问过之后搞到了一个宜昌的金牌客服电话，边打边问地方，两个小时后……
　　“你这个是主板问题，你要修的话我给你报价。”，“恩，报价？啥意思？”，“就是告诉你要修的话多少钱！”，“呃……”，“不含税的话要2500+”，我连忙问“那含税呢？”，“2940”，“……”
　　第二天……
　　第二天，捧着昨晚发的一小笔钱，我又坐上了市里的公汽。
　　“你好，我的笔记本坏啦。”，“是在我们这里买的？”，“哦，不是，我在石家庄买的。”，“哦，什么情况呢？过保了没？”，“刚过了两个个月，其实吧，我听说诺基亚的手机挺耐用的，于是就买了一个，过了一年零一个月，挂了；后来听说Hp的本子挺不错的，质量也还行，就买了一个，到今天刚好一年零两个月，挂了。”，“……”
　　“Hp在我们宜昌这边有金牌服务，你去哪里看一下。”，“俺去了，说是主板坏了，换一块要3000，按电脑买的时候才5000，现在买新的也才3、4K，你说俺修一下划算吗？”，“哦，那也是哈，那你说怎么办呢，修一下是不行得了，看你的意思了。”
　　“俺的意思，俺很正常的用Hp的本子，唯独说俺有过错的话就是俺加了一条内存，但这个内存是从Hp指定宜昌零售店买的，而且是原厂的。所以，你说俺很正常的用它，但是它刚过保就坏，Hp的主板只能支持一年？”，“那肯定不是啦，我建议你在去金牌维修哪里看下，或许诊断错了呢。”
　　一阵纠结之后，我上了电脑城。
　　未完待续……

我想，我是在等一个人

我想，我是在等一个人
她有着清澈的眼睛
明亮的心情

我想，我是在等一个人
她的内心和外表一样的善良
最是那一低头的温柔

我想，我是在等一个人
她有着健康的肤色
而不会像一些人对阳光也充满了厌恶

我想，我是在等一个人
她会和我一起去享受生活的快乐
而不是让烦恼充斥着生活

我想，我是在等一个人
她会在暴雨来临之前买光阿姨的蔬菜
顺便在路上想好第二天的菜肴

我想，我是在等一个人
样式再旧的衣服
在她身上也能找到往日的辉煌

是的，我是在等一个人
我有着坚定的眼神
温暖的心情

是的，我是在等一个人
在我内心中充盈的情感
就等着为你而放

是的，我是在等一个人
我一直坚持用健康的体魄
在阳光下肆意的寻找快乐

是的，我是在等一个人
尽管没有能力奢侈的挥毫
但在一起为日子忙碌却来的久也来的长

是的，我是在等一个人
我会用我最坚实的胸膛
来做她身后永远的依靠

是的，我是在等一个人
在未等到她之前
要知道，我的情感只能为一人而放

写在离512半个多月后

发表于 2008-06-09 | 分类于 认真生活

　　转眼间，距离5.12已经是半个多月了。半个多月的时间，也许我们还在隐约惊觉过的太快，但是对于千万的灾区人民来说，也许这是他们这辈子过的最为漫长的半个多月。在这半个多月里，他们经受着无尽的家园尽毁、流离失所之悲，妻离子散、家破人亡之痛，不知道还要经过多少漫长的时间，他们才能重建起自己的美好家园、重组起自己的温馨家庭、散尽心中的恐惧和悲恸。但在这半个多月里，十三亿的中国人也一样在痛，因为我们是同胞，因为我们是亲人，因为我们都有一个共同的名字——炎黄子孙。

　　半个多月里，每天看到媒体上的灾区资讯，感同身受着灾区人民的悲痛的同时，相信我们每个人都会在想：我们能做些什么？我们还能做些什么？我们能不能再多做些什么？也许作为个人，我们早已捐过款、献过血、为灾区人民献上过祝福和祈祷，我们都做了自己力所能及的事情。但是，看到遇难者数字的不断攀升和灾区接连不断的各种险情，我们都深深知道，这场灾难，真的是超乎我们想象的深重。在这之前，我们根本无法想象会有如此惨重的灾难降临在中华大地上！那一组组不断攀升的数字，那不是单纯的数字啊，那是我们亲爱的同胞们曾经活生生的生命！是曾经与我们一起平凡而又幸福地生活在同一片国土之上的亲人们！每每想到这些，我们在为自己能够平安地活着、安宁地生活而倍感幸福的同时，更觉得自己能为灾区人民做的真的很少，自己所贡献的那份力量也许真的是微薄之极。

　　也许我们所做的一切，对于灾区人民来说都只是杯水车薪，但是我们尽自己所能，最起码，我们可以给千万的灾区人民传达这样一个坚定的声音：我们，一直与你们同在；我们，一直都在你们身后！

节目，是做给听众的

发表于 2008-04-26 | 分类于 快乐工作

今天回到家很累，依然停电，我打开了FM105.9～

　　熟悉的声音，熟悉的节目，格林在节目下和节目里带给我的是完全不同的感受。
　　好久没有在黄昏下直到漆黑、安静的环境中听收音机了，听了一个小时我发现满身的疲惫全都一扫而空，现在的我又精神百倍了。
其实我是想说：

  1. 我们的节目是做给听众的，我们做节目不是要自己做的开心，自己做的有感觉，是让听众开心，听众有感觉。
  2. 我们的音质、音色乃至语言都不是我们节目里最重要的，最重要的是我们想要向听众表达的内容。但，我们的语言、音质音色以及普通话却是最最基本的，是承载内容的基石。
  3. 我们的语言听众能不能接受？要知道，语言是一个人思维的体现，它反映的是一个人的思维方式。
  4. 你的语言，或者说你的思维方式听众不能接受，那么就要改变它，自己去练习发散思维，这是一个不错的方法。
  5. 在态度上，我们在平时要从容地面对生活，这种态度放在节目上也是尤为重要。
  6. 还是要重复的给自己说：一定要注意说话的对象，听众就在你话筒前三米的地方，面对着他，真诚的说话。
  7. 说到真诚，最后一点，要待人真诚。你在节目里面是否真的在真诚地说话，用心的感受，听众都可以感觉得到。所以，在平日里你是一个什么样的人，会做什么样的事，都会在节目中表现出来。
