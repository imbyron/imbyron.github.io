心灵感应

发表于 2011-10-04 | 分类于 认真生活

　　你相信心灵感应的存在吗？其实不管你信不信，我是信了。

　　在一次表演上，有俩外国魔术师——杰克兄弟为大家表演了扑克牌的心灵感应。首先出场的是老杰克，只见他拿出一副单车扑克牌，去掉大小王之后在现场任意找了一位观众，随机抽出五张牌，老杰克用眼睛迅速扫了一下五张牌，分别是红桃4，梅花6，方片A, 黑桃5和方片9，然后选出方片A交给观众，让观众将牌翻过去，自己将剩下的四张牌正面朝上一一摆放到桌面上，而刚刚发生的这一切，站在远处的小杰克完全无法看清楚。之后，老杰克对这位观众说，“刚才我给你的这张牌是什么，天知、地知、你知、我知，但是我和我的弟弟小杰克在心理上有一种神秘的感应力量，他也可以猜得到这张牌是什么，下面就请小杰克走过来”。小杰克走到放着扑克牌的桌子前，双手合十放到胸前，沉思几秒钟，迅速说出了老杰克给观众的牌是方片A。

　　这个魔术从上个世纪八十年代开始在美国流行，很多观众看完魔术后第一反应是老杰克和小杰克之间一定发生了某种信息交流，比如手势呀、身体语言呀等等，可是用一两个动作同时表示出一张牌的花色和大小并非易事。而且有些时候为了增加魔术的趣味性，一些魔术师在表演这个魔术时，两个表演搭档身处两个城市甚至身处两个国家，通过电话、电子邮件、彩信等来完成这个表演，其实这个魔术是一个纯粹的数学游戏，最初由美国麻省理工学院一个研究数学的博士发明。不过无论表演的方式如何改变，有一点是不变的，小杰克必须知道或看到五张牌中剩下的四张牌。

　　既然魔术的玄机在在这四张牌上，那么老杰克是如何通过这四张牌告诉小杰克第五张牌的花色和大小的呢？

ubuntu中怎样去掉每次开机都出现的密钥环

发表于 2010-10-01 | 分类于 爱折腾

　　用Ubuntu后有一件事情一直困扰我，因为我用的无线路由器，所以每次开机连接无线网络都要询问“密钥环密码”。

　　在论坛上搜索过一些解决方法，但是都要删除密码云云……让人心不是很静，现在好了：

点击面板系统托盘上的无线网络图标的“编辑连接”；

在“无线”选项卡里，选择“编辑”；

选中左下角的“对所有用户可用”的选项，（以后不用再输入密钥环密码了）

再点击应用，会提示输入用户登录密码以授权；

然后点击无线网络图标，点击无线网络的名字，即可自动连接至无线网络；

以后每次开机，都不会再提示输入密匙环才能连接无线网络了。

If you want to build a ship

发表于 2010-09-09 | 分类于 快乐工作

　　今天早上无意中翻开了以前的老博客，看到了以前自己写的一篇文章，现在拿到这里来再次的勉励自己。

　　下面这段英文在我印象中好像是某一个古国的谚语，说的很有道理：

If you want to build a ship

don’t herd people together to collect wood

and don’t assign them tasks and work,

but rather teach them to long for the

endless immensity of the sea.

如果你想造一艘船

先不要雇人去收集木头

也不要给他们分配任何任务

而是去激发他们对海洋的渴望
