目前常用的Alfred功能及插件

发表于 2017-09-04 | 分类于 爱折腾

记录一下 Mac 上的一个堪称「神器」的效率软件——Alfred。

Alfred 是一个用键盘通过热键、关键字、自定义插件来加快操作效率的工具。甚至有人在使用了 Alfred 之后丢掉了鼠标。

我常用的功能：

  1. 输入 APP 名称中的关键字打开 APP，比如「au」是打开「Audition」，「ph」可以迅速打开「Photoshop」等。
  2. 输入文件夹名称或文件名称迅速打开（回车）或预览（shift）文件。
  3. 输入「in」+ 关键字来查找文本文件内含有查关键字的文件。
  4. 通过「option」+ 「↑」来将选中的文件夹或者文件加入到缓存。
  5. 通过「option」+ 「→」来批量处理缓存中的文件夹和文件。
  6. 快速在需要的搜索引擎搜索关键词。
  7. 计算器。
  8. Clipboard 粘贴板功能，我自己设置的双击「Ctrl」打开，可以浏览近期「Command+C」的所有内容。
  9. 配合搜狗输入法的「自定义短语」使用，比如我要写博客，需要在终端进入我的博客目录，懒得配置终端的一些命令，于是就把> cd Documents/MyNuts/blog放入了搜狗输入法的「自定义短语」中，要写博客的时候，激活 Alfred，输入boke（我设置的快捷键）回车即可激活终端并进入博客目录。

常用的workflow 插件：

Alfred 杀手级功能 WorkFlows，可以用别人写好的脚本或自己动手，写适用于自己的自动化工作流。

下面是我常用的插件：

  1. 人民币金额大写，输入金额，自动转换为大写，回车即自动复制到粘贴板。
  2. pm2.5 查询，输入城市即可查询该城市的 pm2.5。
  3. 百度地图，可迅速激活百度地图进行关键字的查询。
  4. Domain 查询，可迅速查询一个域名是否被注册，进行域名头脑风暴的时候很方便。
  5. IP 查询，快速查询一个 IP 的归属（调用的淘宝 IP 地址库）。
  6. 天气查询，快速查询一个地方的天气情况。
  7. 「真相」小工具，快速生成类似「我(bu)挺(yao)好(re)的(wo)」的短语。

我 Mac 上在用的三个触发角：

由于我目前用的黑苹果木有触控板，所以有些功能我把它设置在了触控角上。

  1. 左上角，打开 launchpad，显示所有 APP 图标。
  2. 右上角，Mission Control，显示正在用的所有程序。
  3. 右下角，显示桌面。

这样，经过配置，稍稍释放了点没有触控板带来的失落感。
