ubuntu中怎样去掉每次开机都出现的密钥环

发表于 2010-10-01 | 分类于 爱折腾 |

　　用Ubuntu后有一件事情一直困扰我，因为我用的无线路由器，所以每次开机连接无线网络都要询问“密钥环密码”。

　　在论坛上搜索过一些解决方法，但是都要删除密码云云……让人心不是很静，现在好了：

点击面板系统托盘上的无线网络图标的“编辑连接”；

在“无线”选项卡里，选择“编辑”；

选中左下角的“对所有用户可用”的选项，（以后不用再输入密钥环密码了）

再点击应用，会提示输入用户登录密码以授权；

然后点击无线网络图标，点击无线网络的名字，即可自动连接至无线网络；

以后每次开机，都不会再提示输入密匙环才能连接无线网络了。

If you want to build a ship

发表于 2010-09-09 | 分类于 快乐工作 |

　　今天早上无意中翻开了以前的老博客，看到了以前自己写的一篇文章，现在拿到这里来再次的勉励自己。

　　下面这段英文在我印象中好像是某一个古国的谚语，说的很有道理：

If you want to build a ship

don’t herd people together to collect wood

and don’t assign them tasks and work,

but rather teach them to long for the

endless immensity of the sea.

如果你想造一艘船

先不要雇人去收集木头

也不要给他们分配任何任务

而是去激发他们对海洋的渴望

快乐的阅读时光

发表于 2010-09-08 | 分类于 认真生活 |

　　在宜昌等待面试的这些日子，突然怀念起了以前大一在图书馆无忧无虑的时光。

　　图书馆现在布置的比以前多了几分情调，除去一大片墨绿色的书桌外，剩下的空地居然被一堆竹子编成的桌椅占满，椅子坐起来非常的舒服，只可惜桌子于我来说矮了一些。我拿了一杯清茶坐在靠窗的位置，窗外的树叶就这样在我眼前晃动着，伸手就可以抓到。隔着树叶朝对面望去，对面是园丁们修正的很整齐的草丛，草丛边缘是我从来没有去过的档案室，门前的苔藓旁边有一朵黄色的小花儿，对着图书馆怒放，再靠旁边是一些不知名的花草，也正盛开。

　　我坐在椅子上，又放松向下滑了一点找了一个舒服的姿势，将双脚向前伸开，翻开一本小说来读。旁边的人看起来和三年前是没多大分别的，表情神态也都各异：有冥思苦想状、含笑阅读状，还有的在用心的讨论问题。我手上拿的这本书是关于心理学的，以前对心理学总是有一种莫名的排斥，总以为那是一门很暗淡的学科，其实是自己没有接触它罢了，后来自从我读了几个心理学的小游戏后，对心理学的看法便有了些许转变。有朋友说：“学心理学的人能看透别人的心理，知道他人心里想什么，好厉害！”这些说法把心理学神化了，其实，心理学就是一门研究人的心理活动的规律的科学。心理学一词其实来源于希腊文，意思是关于灵魂的科学。灵魂在希腊文中也有气体或呼吸的意思，因为古代人们认为生命依赖于呼吸，呼吸停止，生命就完结了。

　　书中倒是有一些心理学方面的引文词汇，我虽然都能很好的拼读出来，但是其意思对我来说就云里雾里了。我很遗憾自己到现在还没有学好英语，其实自己并不是太排斥它，我想没学好的原因一方面是懒惰，一方面其实还是懒惰。因为英语不好，我前段时间甚至错失了一份很好的工作，这个是我的一个遗憾，学好英语是我正式工作后的一个目标，青春已经所剩无多，我希望我不要再给自己留遗憾了。

　　塞上了耳机，找出一些自己喜欢的音乐，伴随着我的阅读，一杯茉莉花茶也就渐渐的见底了。不用担心考试，不用担心上课，轻松的阅读，还有我最爱喝的茉莉花茶——这便是我在宜昌过的最快乐的时光了。

　　现在我买上了到杭州的车票，虽然舍不得宜昌，但还是要离开，在离开宜昌之前，我将这段时间在图书馆的一点思绪记录下来，用于怀念。
