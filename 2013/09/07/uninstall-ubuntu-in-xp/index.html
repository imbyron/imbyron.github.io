在xp下安全删除ubuntu系统

发表于 2013-09-07 | 分类于 爱折腾 |

准备出掉X100e，自然，把Ubuntu也一并的喀嚓掉会对买家好一些。

　　不过现在删掉Ubuntu不能那么简单得直接格了扇区，不然XP就进不去了，这是因为装了Ubuntu后，系统就由grub引导了，而格掉Ubuntu会使grub损坏，需要先把引导程序换成XP下的引导程序才行。

　　这里有一个工具：百度网盘

　　MBRFix，这个工具，下载下来后解压掉，对应XP系统的是那个不带64的。把它解压到C盘根目录，然后开始>>>运行>>>cmd，输入下面的命令就可以了：

C:\>mbrfix /drive 0 fixmbr /yes

　　然后重启电脑发现grub引导程序不见了，直接进入了XP系统。这时只需要找任意一种方法格式化那个安装Ubuntu的扇区即可。
