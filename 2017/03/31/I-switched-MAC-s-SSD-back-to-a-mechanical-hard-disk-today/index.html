我今天拆了 Mac 的 SSD 又换回了机械硬盘

发表于 2017-03-31 | 分类于 爱折腾

我有一台服役将近 6 年的 Mac：MacBook Pro (13-inch, Early 2011)

　　当年的 Mac 标配是没有 SSD 的，SSD 的好处看得我心痒，总想入一个，但在当时 1400 大洋的价格却令人不忍下手，但最终还是入手了。
　　
　　自从 12 年我把 Mac 的硬盘换成 SSD 之后，各种程序开启时间就短了许多，但是 Mac 也出了一些意料之外的问题，比如，动不动就卡死，没有了之前的如丝般顺滑，一直怀疑是镁光 SSD 的问题，也一直没有证据。就这样断断续续，一直到昨天，我要重新启用我的 Mac 作为工作的主力机了，终于下定决心把 Mac 换回到自带的机械硬盘。
　　
　　不负众望，装完系统和各类软件后，终于在 Mac 上重新体验到了之前的如丝般顺滑，罪魁祸首找到了，心里一块石头落了地，我的大 Mac 终于可以再战三年！

Mac 安装的软件

  • 网易云音乐：随便听听，现在听歌时间其实也很少
  • 坚果云：不同电脑间同步盘，良心应用，买过几个月的专业版。
  • 多态：畅游互联网专用工具。
  • 微信：不解释。
  • QQ：新版还不错。
  • OmniFocus：GTD 工具，管理长期目标和计划
  • Keynote：吃饭的工具，用它做交互、原型、幻灯片。
  • MindNote：Mac 上最好用的思维导图工具。
  • Things：标准的 GTD 工具，打开速度比 OmniFocus 快很多，使用频率高。
  • iA Writer：这篇文章就是用它写的，比 Byword 设计好看多了。
  • Ulysses：我用它管理成体系的文档。
  • Folx PRO：用起来很妙的一个下载工具，比迅雷好。
  • Final Cut Pro：学习中。
  • Adobe Audition CC：吃饭的工具，制作音频的专业软件。
  • Adobe Photoshop CC：杀鸡也用牛刀。
  • WebStorm：他们家的 IDE 做的都很不错，有钱了入全家桶。
  • Sublime Text：性能十分优良的编辑器，简单写一些 Python 小脚本。
  • Dash：它除了可以管理各类的 API 还可以管理 Sheet 这种事情我会乱说？
  • Tweetbot：让 Twitter 飞啦！
  • Tower：图标很丑，Git 图形化工具。
  • 1Password：五颗星，没谁也不能没有它。
  • Dropbox：配合 1password 同步用，对，只用来同步 1password。
  • parallels desktop：神器，Mac 上最好的虚拟 windows 的应用。
  • Alfred 3：其实我用它的工作流用的不多，都是一些简单的用法。
  • PopClip：重度使用，下文会介绍我常用的插件。
  • BetterZip：一直在用的解压缩软件。
  • Python：Mac 自带的是 python2，我下载了 python3.
  • hexo：我用它来生成博客。
  • Node.js：hexo 是用它写的，必须要有它。

Sublime Text 的插件、主题和配置

插件：

  • Package Control：管理插件的插件，最先安装。
  • A File Icon：文件名前面会出现设计非常精美的小图标。
  • AutoFileName：自动完成文档中插入的文件名及后缀。
  • ColorPicker：拾色器。
  • Emmet：写 HTML 利器
  • JsFormat：JS 格式化插件。
  • SideBarEnhancements：丰富侧边栏的插件。
  • SublimeCodeIntel：支持多种语言的各种提醒和补全
  • SublimeREPL：在 sublime 中直接运行 python、ruby 等脚本。

主题：Material Theme

PopClip 常用插件推荐

　　PopClip 是一个极其好用的 Mac 上的增强工具，Google 一下，很多站点，很多篇文章都有介绍它的专题，我在这里就不多浪费笔墨了，下面列出我推荐的一些插件。 　　

  • Paste and Match Style：复制一些文字，长摁呼出功能键，粘贴无格式文字。
  • Title Case：按照英文文法的规范，把一句英文标题该大写的字母变为大写。
  • Quotes：将选中的文字用引号引起来，我一般用它来实现输出「中文引号」。
  • Large Type：大爱的一个插件，将选中的文字变大至全凭，讲课的时候，给人看东西的时候常用。
  • Google Translate：将选中的问题用 Google web 翻译，翻译为中文。
  • Things：将选中的文字发送至 Things 的 inbox 收集箱，很好用的一个插件。
  • Google：将选中的文字发送至 Google 搜索。
  • Wikipedia：将选中的文字发送至 Wikipedia 搜索，可以选择语言。
  • Tweetbot：将选中的文字用 Tweetbot 发一条 Twitter。
  • Shorten Link：将长链接一件缩短——https://is.gd/cRr43j
  • Dash：将选中的文字在 Dash 中进行查询。

更多插件，大家可以到官网进行浏览下载。
